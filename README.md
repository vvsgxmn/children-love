# 基于Spring Boot+Vue 3的留守儿童关爱平台

![Spring Boot](https://img.shields.io/badge/-Spring%20Boot-6DB33F?style=flat-square&logo=spring-boot&logoColor=white) ![Vue3](https://img.shields.io/badge/-Vue3-grey?style=flat-square) ![ElementPlus](https://img.shields.io/badge/-ElementPlus-grey?style=flat-square) ![MySQL](https://img.shields.io/badge/-MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white) 

## 项目介绍

一个综合性的留守儿童关爱服务平台，旨在通过信息化手段整合各方资源，为留守儿童提供全方位的关爱服务；系统覆盖儿童档案建立、志愿者活动组织、爱心物资捐赠、心理咨询辅导、论坛互动交流等场景，帮助管理者高效掌握留守儿童现状，协调社会力量参与关爱行动。

## 功能模块

- **儿童信息管理**：录入和维护留守儿童基本信息，支持按姓名、地区搜索筛选
- **志愿活动管理**：发布志愿活动，志愿者报名参与，活动状态跟踪
- **爱心捐赠管理**：支持资金和物资两种捐赠类型，记录捐赠明细和受助儿童
- **关爱记录管理**：记录每次关爱帮扶详情，形成完整关爱档案
- **心理咨询管理**：在线预约心理咨询，记录咨询内容和反馈
- **就医记录管理**：跟踪儿童就医情况，建立健康档案
- **就学档案管理**：记录儿童教育信息和学习情况
- **健康分析**：对儿童健康数据进行统计分析
- **论坛社区**：分类帖子发布，支持置顶、评论互动，热门帖子排行
- **紧急呼救与救援任务**：紧急信号管理和救援任务分配
- **平台公告**：管理员发布系统公告通知
- **用户管理**：多角色管理（管理员、志愿者、监护人），权限区分

## 技术架构

- **后端**：Spring Boot 2.4.0 + MyBatis-Plus 3.4.1 + JWT 认证
- **前端**：Vue 3.5 + Element Plus 2.8 + Vite 7 + ECharts 6 + Pinia 3
- **数据库**：MySQL 8.0
